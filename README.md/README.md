# StayEasy Hotel Reservation System

Aplikasi reservasi hotel sederhana berbasis PHP MVC
menggunakan PostgreSQL.

## Fitur
- Daftar kamar
- Reservasi kamar
- Simpan data ke database
- Struktur MVC

## Teknologi
- PHP Native
- PostgreSQL
- Laragon

## Cara Menjalankan
1. Jalankan Laragon
2. Aktifkan PostgreSQL
3. Import database/stayeasy.sql
4. Buka browser:
   http://localhost/Project_StayEasy/public/
